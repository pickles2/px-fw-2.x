このディレクトリに配置されているファイルは、コンテンツのサンプルコード集です。

サイトマップCSV

これらのサンプルコードは、サイトマップCSVに定義されています。サイトマップCSVは次のパスに配置されています。

  • ./px-files/sitemaps/sitemap.csv

このファイル3行目以降は、サンプルページについて定義しています。

この定義と見比べながら、サイトマップの設定がページにどのように影響するか、ブラウザで確認しながら見比べてみてください。
サイトマップの値を変更してみながら、画面の変化を観察すると、より理解しやすいと思います。

HTMLコンテンツの書き方

コンテンツエリアのみHTMLを書くようにします。

<p>
	コンテンツ本文のHTML部分。<br />
</p>

コンテンツにスタイルシートを追加する

コンテンツに特有のCSSを書きたい場合など、headセクション内にコードを出力したい場合があります。

そのような場合は、次の例のように、$px->bowl()->send( $html, 'head' ) にコードを渡してください。

<?php ob_start(); ?>
<style type="text/css">
.cont_exam{
	font-size:90%;
	color:#888888;
}
</style>
<?php $px->bowl()->send( ob_get_clean(), 'head' ); ?>

<p class="cont_exam">
	コンテンツ本文のHTML部分。<br />
</p>
