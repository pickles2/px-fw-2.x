PX Commands "api"

[API] api.get.*

PX=api.get.version
Pickles Framework のバージョン番号を取得します。
PX=api.get.config
設定オブジェクトを取得します。
PX=api.get.sitemap
サイトマップ全体の配列を取得します。$px->site()が利用できない場合、falseを返します。
PX=api.get.page_info&path={$path}
$px->site()->get_page_info({$path}) の返却値を取得します。$px->site()が利用できない場合、falseを返します。
PX=api.get.parent
$px->site()->get_parent() の返却値を取得します。$px->site()が利用できない場合、falseを返します。
PX=api.get.actors
$px->site()->get_actors() の返却値を取得します。$px->site()が利用できない場合、falseを返します。
PX=api.get.role
$px->site()->get_role() の返却値を取得します。$px->site()が利用できない場合、falseを返します。
PX=api.get.children&filter={$filter}
$px->site()->get_children() の返却値を取得します。$px->site()が利用できない場合、falseを返します。
PX=api.get.bros&filter={$filter}
$px->site()->get_bros() の返却値を取得します。$px->site()が利用できない場合、falseを返します。
PX=api.get.bros_next&filter={$filter}
$px->site()->get_bros_next() の返却値を取得します。$px->site()が利用できない場合、falseを返します。
PX=api.get.bros_prev&filter={$filter}
$px->site()->get_bros_prev() の返却値を取得します。$px->site()が利用できない場合、falseを返します。
PX=api.get.next&filter={$filter}
$px->site()->get_next() の返却値を取得します。$px->site()が利用できない場合、falseを返します。
PX=api.get.prev&filter={$filter}
$px->site()->get_prev() の返却値を取得します。$px->site()が利用できない場合、falseを返します。
PX=api.get.breadcrumb_array
$px->site()->get_breadcrumb_array() の返却値を取得します。$px->site()が利用できない場合、falseを返します。
PX=api.get.dynamic_path_info&path={$path}
$px->site()->get_dynamic_path_info({$path}) の返却値を取得します。$px->site()が利用できない場合、falseを返します。
PX=api.get.bind_dynamic_path_param&path={$path}¶m={$param}
$px->site()->bind_dynamic_path_param({$path}, {$param}) の返却値を取得します。{$param} には、パラメータのキーと値の組を必要分格納したオブジェクトをJSON形式の文字列で指定します。$px->site()が利用できない場合、falseを返します。
PX=api.get.path_homedir
*[非推奨]* このAPIは PX=api.get.realpath_homedir に改名されました。 古い名前も残されていますが、使用は推奨されません。
PX=api.get.realpath_homedir
$px->get_realpath_homedir() の返却値を取得します。
PX=api.get.path_controot
$px->get_path_controot() の返却値を取得します。
PX=api.get.path_docroot
*[非推奨]* このAPIは PX=api.get.realpath_docroot に改名されました。 古い名前も残されていますが、使用は推奨されません。
PX=api.get.realpath_docroot
$px->get_realpath_docroot() の返却値を取得します。
PX=api.get.path_content
$px->get_path_content() の返却値を取得します。
PX=api.get.path_files&path_resource={$path}
$px->path_files({$path}) の返却値を取得します。$px->site()が利用できない場合、falseを返します。
PX=api.get.realpath_files&path_resource={$path}
$px->realpath_files({$path}) の返却値を取得します。$px->site()が利用できない場合、falseを返します。
PX=api.get.path_files_cache&path_resource={$path}
$px->path_files_cache({$path}) の返却値を取得します。$px->site()が利用できない場合、falseを返します。
PX=api.get.realpath_files_cache&path_resource={$path}
$px->realpath_files_cache({$path}) の返却値を取得します。$px->site()が利用できない場合、falseを返します。
PX=api.get.realpath_files_private_cache&path_resource={$path}
$px->realpath_files_private_cache({$path}) の返却値を取得します。$px->site()が利用できない場合、falseを返します。
PX=api.get.domain
$px->get_domain() の返却値を取得します。
PX=api.get.directory_index
$px->get_directory_index() の返却値を取得します。
PX=api.get.directory_index_primary
$px->get_directory_index_primary() の返却値を取得します。
PX=api.get.path_proc_type
$px->get_path_proc_type() の返却値を取得します。
PX=api.get.href&linkto={$path_linkto}
$px->href({$path_linkto}) の返却値を取得します。$px->site()が利用できない場合、falseを返します。
PX=api.get.canonical&linkto={$path_linkto}
$px->canonical({$path_linkto}) の返却値を取得します。$px->site()が利用できない場合、falseを返します。

[API] api.is.*

PX=api.is.match_dynamic_path&path={$path}
$px->site()->is_match_dynamic_path({$path}) の返却値を取得します。
PX=api.is.page_in_breadcrumb&path={$path}
$px->site()->is_page_in_breadcrumb({$path}) の返却値を取得します。
PX=api.is.ignore_path&path={$path}
$px->is_ignore_path({$path}) の返却値を取得します。
